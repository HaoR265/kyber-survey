# Basics（从零到入门）

本目录记录进入 ML-KEM / Kyber 之前的最小必备基础，目标是：能顺畅阅读 FIPS 203、Kyber 规格书与关键论文。

## 建议顺序
1. 密码学最小背景：KEM / PKE / CPA-CCA / Hash 的作用位置
2. 线性代数最小背景：向量/矩阵、模运算、线性映射的直觉
3. 概率最小背景：统计距离、尾界、失败概率
4. 格与模格最小背景：格、对偶、LWE/Module-LWE 的直觉（不追求证明）

## 产出形式
- 每个主题一份 Markdown：`topic-xxx.md`
- 术语表：`glossary.md`
- 常用公式/符号表：`notation.md`
